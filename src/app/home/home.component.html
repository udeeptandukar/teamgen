Pair generator is able to generate random teams of pairs which can help distribute team members for working in groups or do pair programming.


Add members from which pairs will be generated using member-add command. Members should be provided separated by space. Subsequent call to member-add command will replace the previous member list.

/pair-generator member-add Diana Clark Kara Bruce Barry


If some members should not be paired with each other, use member-exclusion command. Members should be provided separated by space. Subsequent call to member-exclusion command will replace the previous member list.

/pair-generator member-exclusion Diana,Clark Kara,Clark


To see configuration of pair generator, type /pair-generator show-config command.

Configuration from pair generator


To generate random teams of pairs, use /pair-generator generate command.
