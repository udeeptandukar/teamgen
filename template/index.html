Pair generator is able to generate random teams of pairs which can help distribute team members for working in groups or do pair programming.


Add members from which pairs will be generated using /pair-generator member-add command. Members should be provided separated by space. Subsequent call to /pair-generator member-add command will replace the previous member list.


If some members should not be paired with each other, use /pair-generator member-exclusion command. Members should be provided separated by space. Subsequent call to /pair-generator member-exclusion command will replace the previous member list.


To see configuration of pair generator, type /pair-generator show-config command.

Configuration from pair generator


To generate random teams of pairs, use /pair-generator generate command.
